# Live Playground

Interactive code examples you can edit and run directly in your browser.

<Callout type="info">
Examples run against **Conflux eSpace Testnet** by default. Use the network toggle to switch to a local node (`localhost:8545`).
</Callout>

## Available examples

- [Read Balance](/examples/read-balance) — Query native CFX and ERC-20 balances
- [Deploy ERC-20](/examples/deploy-token) — Deploy a full-featured token from the bootstrap library
- [Send CFX](/examples/send-cfx) — Construct and sign a native transfer
