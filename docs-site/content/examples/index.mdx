# Live Playground

Interactive code examples you can edit and run directly in your browser.

<Callout type="info">
Examples run against **Conflux eSpace Testnet** (`chainId: 71`). Sandpack runs inside a remote sandbox and cannot reach `localhost`, so only public RPC endpoints are supported.
</Callout>

---

## Available examples

### @cfxdevkit/core · eSpace client

- [Block Number](/examples/espace-block-number) — Connect `EspaceClient`, read block number, gas price, and latest block header
- [Read Balance](/examples/read-balance) — Query native CFX and ERC-20 balances
- [ERC-20 Token Info](/examples/erc20-info) — Batch-read token metadata with `ERC20_ABI` and `readContract`
- [Send CFX](/examples/send-cfx) — Sign and broadcast a native transfer with `EspaceWalletClient`
- [Deploy ERC-20](/examples/deploy-token) — Deploy a token contract using `deployContract`

### @cfxdevkit/services

- [Swap Quote](/examples/swap-quote) — Get a live WCFX → USDT quote from Swappi DEX
