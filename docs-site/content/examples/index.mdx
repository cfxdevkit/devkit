# Live Playground

Interactive code examples you can edit and run directly in your browser.

<Callout type="info">
Examples run against **Conflux eSpace Testnet** (`chainId: 71`). Sandpack runs inside a remote sandbox and cannot reach `localhost`, so only public RPC endpoints are supported.
</Callout>

---

## Available examples

### @cfxdevkit/core

- [EspaceClient](/examples/espace-block-number) — Connect `EspaceClient`, check connectivity, read block number, gas price, and block details
- [Balances](/examples/read-balance) — Query native CFX and ERC-20 token balances
- [ERC-20 Token Info](/examples/erc20-info) — Batch-read token metadata with `ERC20_ABI` and `client.publicClient.readContract`

### @cfxdevkit/wallet

- [HD Wallet / Mnemonic](/examples/send-cfx) — Generate mnemonics, validate phrases, and derive HD wallet accounts (eSpace + Core addresses)

### @cfxdevkit/contracts

- [Contract ABIs](/examples/swap-quote) — Explore bundled ERC-20/ERC-721/ERC-1155 ABIs and live-read a testnet token
- [Deploy ERC-20](/examples/deploy-token) — Deploy a token using `erc20BaseAbi` + `erc20BaseBytecode` via `wallet.deployContract`
