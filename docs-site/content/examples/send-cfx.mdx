# Send CFX

Construct, sign and broadcast a native CFX transfer on Conflux eSpace.

<Callout type="warning">
This example uses a well-known **test private key** (Hardhat account 0). Replace it with your own **testnet wallet** before sending real transactions.
</Callout>

<Playground
  file="/index.ts"
  files={{
    "/index.ts": `import {
  createPublicClient,
  createWalletClient,
  http,
  parseEther,
  formatEther,
  privateKeyToAccount,
} from 'viem'
import { NETWORK } from './network-config'

// ── Config ─────────────────────────────────────────────────
// ⚠️  Use a TESTNET private key — never mainnet keys in browser code
const PRIVATE_KEY = '0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80'
const RECIPIENT   = '0x70997970C51812dc3A010C7d01b50e0d17dc79C8'
const AMOUNT      = parseEther('0.001')  // 0.001 CFX

// ── Setup ───────────────────────────────────────────────────
const account = privateKeyToAccount(PRIVATE_KEY as \`0x\${string}\`)

const publicClient = createPublicClient({
  transport: http(NETWORK.rpcUrl),
})

const walletClient = createWalletClient({
  account,
  transport: http(NETWORK.rpcUrl),
})

// ── Main ────────────────────────────────────────────────────
async function main() {
  console.log('Network:  ', NETWORK.rpcUrl)
  console.log('Chain ID: ', NETWORK.chainId)
  console.log('From:     ', account.address)
  console.log('To:       ', RECIPIENT)
  console.log('Amount:   ', formatEther(AMOUNT), 'CFX')
  console.log()

  // Check sender balance
  const balance = await publicClient.getBalance({ address: account.address })
  console.log('Sender balance:', formatEther(balance), 'CFX')

  if (balance < AMOUNT) {
    console.log('⚠  Insufficient balance — skipping send')
    console.log('   Fund this address on testnet: https://faucet.confluxnetwork.org/')
    return
  }

  // Estimate gas
  const gas = await publicClient.estimateGas({
    account: account.address,
    to: RECIPIENT,
    value: AMOUNT,
  })
  console.log('Estimated gas:', gas.toString())

  // Send transaction
  const hash = await walletClient.sendTransaction({
    to: RECIPIENT,
    value: AMOUNT,
    chain: null,  // uses wallet's current chain
  })

  console.log()
  console.log('✓ Transaction sent!')
  console.log('  Hash:', hash)

  // Wait for receipt
  const receipt = await publicClient.waitForTransactionReceipt({ hash })
  console.log('  Block:', receipt.blockNumber)
  console.log('  Gas used:', receipt.gasUsed.toString())
}

main().catch(console.error)
`
  }}
  showConsole

## Testnet faucet

To fund a testnet address, visit **https://faucet.confluxnetwork.org/** and enter your address.

## With the cfxdevkit SDK

```typescript
import { ClientManager } from '@cfxdevkit/core'
import { parseEther } from 'viem'

const client = new ClientManager({ network: 'testnet' })
await client.connect()

const hash = await client.evm.walletClient.sendTransaction({
  to: '0xRecipient',
  value: parseEther('0.001'),
})
```
