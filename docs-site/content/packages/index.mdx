## Dependency tree

```
@cfxdevkit/core          ← foundation
    ├── @cfxdevkit/contracts
    ├── @cfxdevkit/wallet
    ├── @cfxdevkit/executor
    ├── @cfxdevkit/services  (→ core)
    ├── @cfxdevkit/protocol  (→ core, services)
    ├── @cfxdevkit/compiler
    ├── @cfxdevkit/devnode
    ├── @cfxdevkit/react     (→ core, React)
    ├── @cfxdevkit/defi-react(→ core, services, React)
    └── @cfxdevkit/wallet-connect (→ wagmi, ConnectKit)
```
